# Skill: Deep Research

## Scope
Comprehensive research on any topic — market analysis, technical evaluation, competitor analysis, trend spotting.

## Research Process
1. **Define the question** — What exactly do we need to know?
2. **Broad search** — WebSearch for overview, recent developments
3. **Deep dive** — Read key sources, extract data points
4. **Cross-reference** — Verify claims from multiple sources
5. **Synthesize** — Summarize findings with actionable recommendations
6. **Document** — Save to relevant project brief or memory

## Tools Available
- **WebSearch** — Search the web for current information
- **WebFetch** — Read specific URLs and extract content
- **Bash** — Run scripts, query APIs, process data

## Output Format
Research results should include:
- **Summary** — 2-3 sentence overview
- **Key Findings** — Bulleted list of important facts
- **Data** — Numbers, statistics, comparisons
- **Sources** — URLs for verification
- **Recommendations** — What to do with this information

## Best Practices
- Always check the date of sources — prefer recent information
- Look for primary sources over secondary reporting
- Note conflicting information rather than ignoring it
- Save research summaries to relevant project briefs
- Update MEMORY.md with key facts that might be needed later
