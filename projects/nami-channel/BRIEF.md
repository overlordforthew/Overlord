# Nami's Meditation Channel — Project Brief

## Overview
YouTube channel for Nami Barden's guided meditation content. Goal: automate the video production pipeline.

## Pipeline
1. Market research (trending meditation topics)
2. Script generation (Claude API)
3. Bilingual content (English + target language)
4. Voice synthesis (ElevenLabs — Nami's voice)
5. Visual content (DALL-E imagery, Veo video clips)
6. Video assembly (FFmpeg)
7. YouTube upload and optimization

## Status
- Planning phase
- Video pipeline skill documented
- Veo integration available for AI-generated video clips

## Tools Available
- Claude Code for script writing
- Veo skill for video generation
- ElevenLabs API for voice synthesis
- DALL-E for imagery
- FFmpeg for video assembly
