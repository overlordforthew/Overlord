OnlyHulls (onlyhulls.com) is getting a full visual redesign. Plan created 2026-03-27.

**Design direction:** Dark-mode-first, OnlyFans-inspired aesthetic. Cyan (#00AFF0) primary on dark charcoal (#0a0a0f). Glassmorphism cards, hover glow effects, playful microcopy throughout.

**Why:** The current design is generic blue/white that undersells the playful "OnlyFans of Boats" brand. Competitors (Boat Trader, YachtWorld) have dated 2005-era UX. OnlyHulls needs to stand out with personality.

**How to apply:** All visual changes only — no backend/API/database changes. 14 files to modify in dependency order starting with globals.css. Key structural fix: extract duplicated header/footer from 5 pages into shared SiteNav + SiteFooter components. Plan file at `/root/.claude/plans/crispy-shimmying-breeze.md`.

**Competitive edge to emphasize in design:**
- $0 commission (vs Boat Trader $149+/listing)
- AI-powered matching (unique to OnlyHulls)
- No broker gatekeeping (vs YachtWorld broker-only)
- Modern, fun UX (vs everything else)

**Brand voice:** "Find Your Perfect Hull", "Show Us Your Hull", "Hull yeah!", "No catfishing. Real boats. Real hulls."
