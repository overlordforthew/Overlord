**Boats Group** (Miami) owns: boats.com, Boat Trader, YachtWorld, Boatshop24, Boats and Outboards, iNautia, Cosas de Barcos, Botentekoop, Annonces du Bateau

Combined: ~300,000+ listings, ~75% of global online boat listings market.

**Class action lawsuit** filed August 2025, Southern District of Florida (Brill Maritime v. Boats Group):
- Alleges monopolistic practices, exclusionary contracts, serial competitor acquisitions
- Dealer costs: ~$1,000/mo (2010s) → $5,128/mo (2024) — 400%+ increase
- Boat Trader alone: $2,900/mo in 2025

**Scraping status:** All 9 Boats Group sites share identical Cloudflare JS challenge protection. Blocked from Hetzner IPs. Bypassed via GitHub Actions (Azure IPs) — proven with boats.com scraper.

**Why this matters for OnlyHulls:** The antitrust lawsuit creates a once-in-a-decade window. Dealers/brokers are angry and actively looking for alternatives. Position OnlyHulls as the post-monopoly option: open platform, zero commission, no lock-in.

**How to apply:** Use in marketing copy ("YachtWorld charges $5,000/mo — we charge $0"), broker outreach, and competitive positioning. Reference the lawsuit when context is appropriate.
