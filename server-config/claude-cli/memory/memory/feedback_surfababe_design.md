Gil explicitly said "I want you to learn this design, we like this surfababe.com design." Use the SurfaBabe website design as a reference for future website projects.

**Why:** Gil considers this a good design baseline for his projects. It represents the visual quality he expects.

**How to apply:** When building or redesigning websites for Gil, reference the SurfaBabe design patterns:
- **Color palette:** Teal (#04595E, #82CDCE), salmon (#BC7C5A), sage (#EBEFEA), cream (#F2DBBF), warm browns
- **Typography:** Fraunces (serif headings), Caveat (handwritten accents), DM Sans/Helvetica Neue (body)
- **Layout patterns:** Full-width hero with bg image, two-column grids, rounded cards, pill-shaped buttons
- **Section structure:** Announcement bar > sticky nav > hero > product grid > video > story > contact > gallery > testimonials > FAQ > newsletter > footer
- **Design touches:** Ocean/nature backgrounds, decorative illustrations, photo cards with info overlays, mosaic photo grids, gradient backgrounds
- **Fonts:** 20px bold nav links, 50px serif headings, Caveat for taglines/subtitles
- **Footer:** Dark teal bg, Caveat cursive tagline, 2-column layout with nav links + contact icons
- **Live reference:** surfababe.namibarden.com (the faithful replica) and www.surfababe.com (the Wix original)
